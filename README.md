# Homebrew Tap

Homebrew formulae for deikka tools.

## Instalacion

```sh
brew tap deikka/tap
```

## Formulas disponibles

| Formula | Descripcion |
|---|---|
| `folder-sync` | Backup incremental de carpetas a disco externo con app de menu de barra |

## Uso

```sh
brew install deikka/tap/folder-sync
```
